Installation procedure

Toolbox Archiving Service

The service can be created using the Toolbox administration web interface. A wizard like procedure takes the user to a complete deployment of the service. Follow these steps:
  • Access the Toolbox Administration Web console by providing administrator credentials. The console is usually accessible via the URL http://<IP>:<PORT>/TOOLBOX
  • Click on "Service Management"
  • Click on "Create a new Service"
  • Open the "Create an Archiving service" panel
  • Provide a service name for the service
  • Select "OGC-05-007r7 version 1.0.0" interface type
  • Select "Standalone" as interface mode
  • Click on "Create Archiving Service"
This will create the service and you will be redirected to the configuration page. You have now to modify the service configuration. Follow the below steps:
  • Move to the end of the page and update the field in the "Configure service variables"
  • Click on the "Configure" button
  • The page is refreshed and a list of operation is shown
  • Select all operations and click on "Create"

ARMS Servlet

The ARMS Servlet can be simply deployed on an host by copying its war file under the /webapps folder. Tomcat will automatically deploy the application. Once deployed, the ARMS Servlet needs to be configured. Please refer to the configuration section for further details.
