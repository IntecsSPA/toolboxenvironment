Delete operation

This REST endpoint will allow a client to delete data and metadata. For a full description of how this operation works, please refer to this section The store operation is accessible at the URL http://<TOMCAT_IP>:<TOMCAT_PORT>/ArchivingServer/service/delete/<ITEMD_ID>
The HTTP method to use for accessing the operation is DELETE

Input Message

The only parameter needed by the delete operation is the identifier through which data and metadata can be identified. This identifier is provided by the <ITEMD_ID> field specified in the endpoint URL.

Output message

No output message is returned by the operation. The status of the HTTP request returns to the client all needed information about the execution.
