Delete processing

The delete processing aims to provide to the client the capability to unpublish data and metadata stored with a call to the store processing. For further information about the operations in general, please read this section. The store processing is accessible at the following URL: http://<TOMCAT_IP>:<TOMCAT_PORT>/TOOLBOX/services/<SERVICE_NAME>
The SOAP Action associated to this processing is ExecuteProcess_retrieve
The delete processing has the following Describe document associated:

Description
