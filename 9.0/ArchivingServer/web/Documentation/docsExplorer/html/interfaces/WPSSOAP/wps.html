WPS operation

The Toolbox Archiving Service implements a standard WPS interface with SOAP bindings. Thus all WPS operation are exposed by the service and can be invoked following the methodologies described in the standard document. The service implements the GetCapabilities and DescribeProcessing operation plus an Execute operation for each processing. The full table of the operations implemented by the service is given below with a description and the SOAP Action. All operations can be invoked through a SOAP 1.1 exchange and are synchronous.

Operation

SOAP Action

Description
GetCapabilities GetCapabilities This operation returns the capabilities document for the service
DescribeProcess DescribeProcess This operation returns the describe document for each of the processings provided by the service (store, retrieve and delete)
ExecuteProcess_store ExecuteProcess_store This operation executes a store operation by parsing the input message and forwarding a proper request to the ARMS Servlet.
ExecuteProcess_retrieve ExecuteProcess_retrieve This operation executes a retrieve operation by parsing the input message and forwarding a proper request to the ARMS Servlet.
ExecuteProcess_delete ExecuteProcess_delete This operation executes a delete operation by parsing the input message and forwarding a proper request to the ARMS Servlet.
