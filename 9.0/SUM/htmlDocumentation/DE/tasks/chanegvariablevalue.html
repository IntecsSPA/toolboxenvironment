Change variable value

Once started, the debug session executes the script until a breakpoint is reached. When this happens, the execution is suspended and the perspective is updated. The executing thread view is updated with the suspension file while the variables view is updated with all engine's variables.

To change the value of a variable, select it and right click on it. A menu is shown. Select "Change value".

A new frame is shown, filled with the current value of the select variable. To modify it, type the new value and click "Ok".
