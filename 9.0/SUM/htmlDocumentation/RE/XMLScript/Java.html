Executing Java code

Tag Name java
Description This tag creates a Java class and executes it. The content of the Java class is provided by the user. The first child tag contains the import statements to add to the Java class. The second child tag, enclosed in a CDATA segment, contains the core of the Java class. No class, method or interface definition can be provided since these statements are iternally handled by Toolbox in order to create a class with a predefined method.
Examples 
<java>
		<imports>
			import java.text.SimpleDateFormat;
			import java.util.Date;
		</imports>
		<code><![CDATA[
			SimpleDateFormat formatter=new SimpleDateFormat("yyMMddHHmmss");
		
			booleanType=true;
			integerType=Integer.MAX_VALUE;
			byteType=Byte.MAX_VALUE;
			shortType=Short.MAX_VALUE;
			longType=Long.MAX_VALUE;
			floatType=Float.MAX_VALUE;
			doubleType=Double.MAX_VALUE;
			characterType='X';
			stringType=formatter.format(new Date());
			]]>
		</code>
	</java>
	  
This tag creates a class that updates some of the variables in the Toolbox Engine.
