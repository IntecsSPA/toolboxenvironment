Archiving data

Tag Name zip
Attributes
Name Type Use Default Description
file xsd:string required   The filename of the archive to create.
Description This tag creates a zip archive . The file name of the created archive is passes with the attribute named file. The content of the archive is given by the child tag
Examples 
<zip file="$zipFileName">
		<dir>
	    	<string>/home/toolbox/data/</string>
		</dir>
</zip>
This tag creates a zip file, whose full path is stored in the zipFileName variable. All files and subdirectories under /home/toolbox/data/ are stored into the zip file.
