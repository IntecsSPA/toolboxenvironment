Arrays

Tag Name arrayLength
Description Returns the length of the array returned by the element enclosed in this element.
Examples 
<arrayLength>
	<variable name="arrayName"/>
</arrayLength>
Returns the length of the array stored in the variable "arrayName".

Tag Name arrayLocation
Description Returns the value stored in the location af a multidimensional array. The first element enclosed in this element must return the array whilst the subsequent elements must return the integer expressions to index the array.
Examples 
<arrayLocation>
	<variable name="arrayName"/>
	<variable name="arrayIndex"/>
</arrayLocation>
Returns the value stored in the mono-dimensional array "arrayName" at the index "arrayIndex". 
<arrayLocation>
	<variable name="arrayName"/>
	<variable name="firstDimensionIndex"/>
	<literal value="2"/>
</arrayLocation>
Returns the value stored in the bi-dimensional array at the location having (value stored in the "firstDimensionIndex", 2) as coordinates

Tag Name arrayType
Attributes
Name Type Use Default Description
dimensions xsd:positiveInteger required
The dimension of the arrays represented by the object of class Class returned by the tag.
Description Returns the object of class Class representing arrays with number of dimensions given in attribute "dimensions" and item type given by the enclosed tag.
Examples 
<arrayType dimensions="4">
	<classLiteral class="java.io.File"/>
</arrayType>
Returns the object of class Class representing the arrays of dimension 4 whose items are objects of class java.io.File.

Tag Name newArray
Description Creates a new monodimensional array and returns it. The item type is Object, so the locations of the created array can contain any Java object (including primitive values). The enclosed element must return an integer expression representing the length of the array to be created.
Examples 
<newArray>
	<literal value="3"/>
</newArray>
Returns a mono-dimensional array of lenght "3".

Tag Name newInitializedArray
Description Creates a new linear array, initializes it and returns it. The item type is Object, so the locations of the created array can contain any Java object (including primitive values). The subsequent elements are used to retrieve the objects used to initialize the array.
Examples 
<newInitializedArray>
	<literal value="2"/>
	<literal value="0"/>
	<literal value="8"/>
	<literal value="7"/>
	<literal value="5"/>
</newInitializedArray>
Creates a linear array of integers initialized to {2, 0, 8, 7, 5} 
<sequence>
	<setVariable name="arrayName">
		<newInitializedArray>
			<literal value="2"/>
			<literal value="0"/>
			<literal value="8"/>
			<literal value="7"/>
			<literal value="5"/>
		</newInitializedArray>
	</setVariable>
	.
	.
	<set>
		<arrayLocation>
			<variable name="arrayName"/>
			<literal value="1"/>
		</arrayLocation>
		<literal value="3"/>
	</set>
</sequence>
A linear initialized array of integers is created and stored in the variable "arrayName". Later the second element (thus at index 1) of such array is assigned with 3.

Tag Name newInitializedTypedArray
Description Creates a new linear array, initializes it and returns it. The first element enclosed in this element must return the Class object representing the type of the array elements. The subsequent elements are used to retrieve the objects used to initialize the array.
Examples 
<newInitializedTypedArray>
	<classLiteral class="java.io.File"/>
	<variable name="aFile"/>
	<variable name="anotherFile"/>
	<variable name="thirdFile"/>
</newInitializedTypedArray>
A linear array of objects java.io.File is created and initialized. The array has length 3.

Tag Name newTypedArray
Description Creates a new multidimensional array and returns it. The first element enclosed in this element must return the Class object representing the type of the array elements. The subsequent elements must return integer expressions representing each dimension of the array to be created.
Examples 
<newTypedArray>
	<classLiteral class="ClassName"/>
	<literal value="30"/>
	<literal value="10"/>
	<atoi>
		<variable name="aString"/>
	</atoi>
</newTypedArray>
Creates and returns a three-dimensional array of objects of class "ClassName". The first dimension has length 30, the second one 10 and the third one the integer given by the conversion into an integer of the string stored in the variable "aString".
