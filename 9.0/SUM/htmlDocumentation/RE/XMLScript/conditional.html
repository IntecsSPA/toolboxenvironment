Conditional Operators

Tag Name and
Attributes
Name Type Use Default Description
shortcut xsd:boolean true
Flag indicating whether the evaluation of the enclosed tags must stop after the first false tag is found (default) or all the tags must be evaluated ("shortcut" set to "false")
Description Returns the boolean and of the boolean values returned by the enclosed tags. If the tag is empty, it returns true, if it contains one tag, its value is returned. If the attribute "shortcut" has "true" value (the default), evaluation stops after the first false tag is found, otherwise all the tags are evaluated.
Examples 
<and/>
Returns "true". 
<and>
	<gt>
		<variable name="variableName"/>
		<literal value="0"/>
	</gt>
</and>
Returns "true" if and only if the value stored in the variable "variableName" is greater than "0". 
<and shortcut="false">
	<gt>
		<variable name="variableName"/>
		<literal value="0"/>
	</gt>
	<gt>
		<variable name="anOtherVariableName"/>
		<literal value="0"/>
	</gt>
</and>
Returns "true" if and only if both the variables named "variableName" and "anOtherVariableName" are greater than 0. Both the variables are evaluated. 
<and>
	<gt>
		<variable name="variableName"/>
		<literal value="0"/>
	</gt>
	<gt>
		<variable name="anOtherVariableName"/>
		<literal value="0"/>
	</gt>
	<isNotNull>
		<variable name="anObject"/>
	</isNotNull>
</and>
If the variable named "variableName" is not greater than 0 evaluation stops and "false" is returned. Otherwise, if "anOtherVariableName" is not greater than 0 evaluation stops and "false" is returned. Otherwise,if the object stored in the variable named "anObject" is not null "true" is returned, otherwise "false" is returned.

Tag Name not
Description If the enclosed element returns an integer, this tag returns the integer obtained by complementing each bit. If the enclosed element returns a boolean, this tag returns the negation.
Examples 
<not>
	<literal value="25"/>
</not>
Returns -26. 
<not>
	<literal value="-25"/>
</not>
Returns 24. 
<not>
	<literal type="boolean" value="false"/>
</not>
Returns "true".

Tag Name xor
Description Returns the boolean xor of the boolean values returned by the enclosed tags. If the tag is empty, it returns false, if it contains one tag, its value is returned.
Examples 
<xor/>
Returns false. 
<xor>
    <literal type="boolean" value="true"/>
</xor>
Returns true. 
<xor>
	<literal type="boolean" value="true"/>
	<literal type="boolean" value="false"/>
	<literal type="boolean" value="false"/>
</xor>
Returns true.
