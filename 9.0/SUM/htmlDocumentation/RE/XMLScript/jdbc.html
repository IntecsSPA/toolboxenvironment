JDBC

Tag Name dbClose
Description Closes a JDBC connection. The connection to be closed is returned by the enclosed element.
Examples 
<sequence>
	<setVariable name="connection">
		<dbConnect driver="com.mysql.jdbc.Driver">
			<variable name="URL"/>
			<variable name="userName"/>
			<variable name="password"/>
		</dbConnect>
	</setVariable>
	.
	.
	.
	<dbClose>
		<variable name="connection"/>
	</dbClose>
</sequence>
At the end of the sequence the connection "connection" is closed.

Tag Name dbCommit
Description Commits every update operation performed on the connection given by the enclosed element, since last commit, last rollback, or connection creation.
Examples 
<sequence>
	<setVariable name="connection">
		<dbConnect autoCommit="false" driver="com.mysql.jdbc.Driver">
			<variable name="URL"/>
			<variable name="userName"/>
			<variable name="password"/>
		</dbConnect>
	</setVariable>
	.
	.
	.
	<dbCommit>
		<variable name="connection"/>
	</dbCommit>
</sequence>
At the end of the sequence a commit is performed.

Tag Name dbConnect
Attributes
Name Type Use Default Description
driver xsd:string The JDBC driver to be used
autoCommit xsd:boolean true Flag indicating whether every update performed on the connection is commited (default) or not ("autoCommit" set to "false").
Description Connects to a DBMS via JDBC. The driver attribute gives the JDBC driver to be used. This driver must be in the classpath. The auto-commit attribute (default true) tells if every update performed on this connection is commited or not. The first child must return the URL to be used for connection. The second child must return the user to be used for connection. The third child must return the password to be used for connection. The return value is a JDBC Connection object.
Examples 
<dbConnect driver="com.mysql.jdbc.Driver">
	<variable name="URL"/>
	<variable name="userName"/>
	<variable name="password"/>
</dbConnect>
Connects to the DBMS at the URL "URL" using the driver "com.mysql.jdbc.Driver". Every update performed is commit. 
<dbConnect driver="com.mysql.jdbc.Driver" autoCommit="false">
	<variable name="URL"/>
	<variable name="userName"/>
	<variable name="password"/>
</dbConnect>
Connects to the DBMS at the URL "URL" using the driver "com.mysql.jdbc.Driver". Commits are not automatically performed.

Tag Name dbExecute
Description Executes the SQL statement returned by the second enclosed element (which may return multiple results), on the connection returned by the first enclosed element. Returns true if the first result is a ResultSet object; false if it is an update count or there are no results.
Examples See dbUpdate

Tag Name dbExecuteUpdate
Description Executes the SQL statement returned by the second enclosed element (which may be an INSERT, UPDATE, or DELETE statement or an SQL statement that returns nothing, such as an SQL DDL statement), on the connection returned by the first enclosed element. Returns either the row count for INSERT, UPDATE or DELETE statements, or 0 for SQL statements that return nothing.
Examples See dbUpdate

Tag Name dbGetNumOfRows
Description Returns the number of rows of the ResultSet given by the enclosed element.
Examples 
<dbGetNumOfRows>
	<variable name="sqlResult"/>
</dbGetNumOfRows>

Returns the number of rows in the Result Set contained in the variable "sqlResult".

Tag Name dbGetColumnCount
Description Returns the column number of the result set given by the enclosed element.
Examples 
<sequence>
	<setVariable name="queryResult">
		<dbQuery>
			<variable name="connection"/>
			<variable name="queryString"/>
		</dbQuery>
	</setVariable>
	<dbGetColumnCount>
		<variable name="queryResult"/>
	</dbGetColumnCount>
</sequence>
Returns the number of columns in the Result Set stored in the variable "queryResult".

Tag Name dbGetColumnIndex
Description Returns the column index, given the name, of the result set given by the first enclosed element. The name is given by the second enclosed element.
Examples 
<sequence>
	<setVariable name="sqlResult">
		<dbQuery>
			<variable name="connection"/>
			<variable name="queryString"/>
		</dbQuery>
	</setVariable>
	<setVariable name="columnIndexOfProductPrice">
		<dbGetColumnIndex>
			<variable name="sqlResult"/>
			<string>productPrice</string>
		</dbGetColumnIndex>
	</setVariable>
</sequence>
The index of the column "productPrice" is stored in the variable "columnIndexOfProductPrice".

Tag Name dbGetValue
Description Returns the string value of a field of current row of the ResultSet givne by the first enclosed element. The field is selected by the second enclosed element, using an integer (index) or a string (name).
Examples 
<sequence>
	<setVariable name="sqlResult">
		<dbQuery>
			<variable name="connection"/>
			<variable name="queryString"/>
		</dbQuery>
	</setVariable>
	<dbNextRow>
		<variable name="sqlResult"/>
	</dbNextRow>
	<dbGetValue>
		<variable name="sqlResult"/>
		<string>productPrice</string>
	</dbGetValue>
</sequence>
Returns the value of the field "productSize" in the first row of the result set stored in the variable "sqlResult". 
<sequence>
	<setVariable name="sqlResult">
		<dbQuery>
			<variable name="connection"/>
			<variable name="queryString"/>
		</dbQuery>
	</setVariable>
	<dbNextRow>
		<variable name="sqlResult"/>
	</dbNextRow>
	<dbGetValue>
		<variable name="sqlResult"/>
		<literal value="2"/>
	</dbGetValue>
</sequence>
Returns the value of the second field in the first row of the result set stored in the variable "sqlResult".

Tag Name dbNextRow
Description Moves the cursor of the ResultSet given by the enclosed element, to the next row, if possible. The result of this tag is boolean value telling if it was possible to move to next row. This tag has to be used before any dbGetValue, since initially a ResultSet has the cursor BEFORE first row.
Examples 
<sequence>
	<setVariable name="sqlResult">
		<dbQuery>
			<variable name="connection"/>
			<variable name="queryString"/>
		</dbQuery>
	</setVariable>
	<dbNextRow>
		<variable name="sqlResult"/>
	</dbNextRow>
</sequence>
The cursor of the result set stored in the variable "sqlResult" is moved to the first row. 
<sequence>
	<setVariable name="sqlResult">
		<dbQuery>
			<variable name="connection"/>
			<variable name="queryString"/>
		</dbQuery>
	</setVariable>
	<if>
		<dbNextRow>
			<variable name="sqlResult"/>
		</dbNextRow>
		<print>
			<string>Query produced results!</string>
		</print>
		<print>
			<string>Query response is empty!</string>
		</print>
	</if>
</sequence>
If the Result set stored in the variable "sqlResult" is not empty (i.e. it is possible to move to the first row) the string "Query produced results!" is printed on video, otherwise the string "Query response is empty" is printed on video.

Tag Name dbQuery
Description Performs an SQL query on the connection given by the first enclosed element. The SQL query is given by the second enclosed element. This tags return a JDBC ResultSet.
Examples 
<sequence>
	<setVariable name="connection">
		<dbConnect autoCommit="false" driver="db.driver">
			<variable name="URL"/>
			<variable name="username"/>
			<variable name="password"/>
		</dbConnect>
	</setVariable>
	<setVariable name="sqlResult">
		<dbQuery>
			<variable name="connection"/>
			<variable name="queryString"/>
		</dbQuery>
	</setVariable>
</sequence>
Performs a query on the connection whose name is stored in the variable "connection" and store the result set in the variable "queryResult".

Tag Name dbSetCursor
Description Moves the cursor to the position (row number) returned by the first enclosed element within the ResultSet given by the second enclosed element. If the row number is positive, the cursor moves to the given row number with respect to the beginning of the result set. The first row is row 1, the second is row 2, and so on. If the given row number is negative, the cursor moves to an absolute row position with respect to the end of the result set. For example, calling the method absolute(-1) positions the cursor on the last row; calling the method absolute(-2) moves the cursor to the next-to-last row, and so on. An attempt to position the cursor beyond the first/last row in the result set leaves the cursor before the first row or after the last row. Returns true if the cursor is on the result set; false otherwise.
Examples 
<if>
  <dbSetCursor>
     <literal value="3"/>
     <variable name="sqlResult"/>
  </dbSetCursor>
  <sequence>
     .
     .
  </sequence> 
</if>
Tries to move the cursor to the third (absolute position) row of the result set stored in the variable "resultSet". If motion was possible, the evaluation of the tag "dbSetCursor" returns "true" and the tag "sequence" is executed. Otherwise, the cursor is not moved within the result set and "false" is returned by the evaluation of the tag "dbSetCursor". The tag "sequence" is not executed.
<if>
  <dbSetCursor>
     <literal value="-3"/>
     <variable name="sqlResult"/>
  </dbSetCursor>
  <sequence>
     .
     .
  </sequence> 
</if>

The behaviour is similar to the above example except for the fact that the cursor is tried to be moved on the last but two row of the result set stored in the variable "resultSet".

Tag Name dbUpdate
Description Performs an SQL update on the connection given by the first enclosed element. The SQL string is given by the second enclosed element.
Examples 
<setVariable name="connection">
	<dbConnect autoCommit="false" driver="db.driver">
		<variable name="URL"/>
		<variable name="username"/>
		<variable name="password"/>
	</dbConnect>
</setVariable>
	.
	.
	.
<dbUpdate>
	<variable name="connection"/>
	<string>CREATE TABLE prices (idx INT NOT NULL, productPrice INT, PRIMARY KEY(idx));</string>
</dbUpdate>
Creates the table "price" in the database of the connection whose name is stored in the variable "connection".
