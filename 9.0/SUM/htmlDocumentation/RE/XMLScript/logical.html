Logical Operators

Tag Name bitAnd
Description Returns the integer obtained performing a bit to bit and operation among each integer returned by the enclosed elements. If only one element is enclosed, it is returned as is, if no element is enclosed, -1 is returned
Examples 
<bitAnd/>
Returns -1. 
<bitAnd>
	<literal value="3"/>
</bitAnd>
Returns 3. 
<bitAnd>
	<literal value="3"/>
	<literal value="2"/>
</bitAnd>
Returns 2. 
<bitAnd>
	<literal value="-1"/>
	<literal value="2"/>
</bitAnd>
Returns 2. 
<bitAnd>
	<literal value="0"/>
	<literal value="2"/>
</bitAnd>
Returns 0.
