Relational Operators

Tag Name eq
Description Returns the boolean value obtained evaluating the "equal" operation between the integer values returned by the two enclosed tags
Examples 
<eq>
	<literal value="1024"/>
	<variable name="VariableName"/>
</eq>
If the value stored in the variable named "VariableName" is equal to 1024 returns "true", else returns "false".

Tag Name isNotNull
Description Returns true if and only if the object returned by the enclosed element is not null.
Examples 
<isNotNull>
	<newObject>
		<classLiteral class="java.io.File"/>
		<parameter>
			<classLiteral class="java.lang.String"/>
			<variable name="filePath"/>
		</parameter>
	</newObject>
</isNotNull>

Returns "true".

<isNotNull>
	<null/>
</isNotNull>
Returns "false".

Tag Name oeq
Description Returns true if and only if the objects returned by the two enclosed elements are the same object, or neither is null and the equals method of the first object returns true with second object as parameter
Examples 
<oeq>
	<string>string</string>
	<string>strong</string>
</oeq>

Returns "false".

		</parameter>

</setVariable>

Tag Name oneq
Description Returns true if and only if the objects returned by the two enclosed elements are not the same object, and one of them is null or the equals method of the first object returns false with second object as parameter
