Shift Operators

Tag Name lShift
Description Returns the integer returned by the first enclosed element, shifting left its bits by an amount given by the second element and filling with zeroes.
Examples 
<lShift>
	<literal value="7"/>
	<literal value="2"/>
</lShift>
Returns 28. 
<lShift>
	<multiply>
		<variable name="aVariable"/>
		<literal value="2"/>
	</multiply>
	<literal value="3"/>
</lShift>
Returns the result of the product among the value stored in the variable "aVariable" and 2 shifted left by 3. 
<lShift>
	<literal value="2"/>
	<literal value="7"/>
</lShift>
Returns 256.

Tag Name rSShift
Description Returns the integer returned by the first enclosed element, shifting right its bits by an amount given by the second element and filling with sign bit.
Examples 
<rSShift>
	<literal value="8"/>
	<literal value="2"/>
</rSShift>

Returns 2.

<rSShift>
	<literal value="2"/>
	<literal value="7"/>
</rSShift>

Returns 0.

<rSShift>
	<literal value="-1"/>
	<literal value="15"/>
</rSShift>
Returns -1.

Tag Name rUShift
Description Returns the integer returned by the first enclosed element, shifting right its bits by an amount given by the second element and filling with zeroes.
Examples 
<rUShift>
	<literal value="8"/>
	<literal value="2"/>
</rUShift>

Returns 2.

<rUShift>
	<literal value="2"/>
	<literal value="7"/>
</rUShift>

Returns 0.

<rUShift>
	<literal value="-1"/>
	<literal value="15"/>
</rUShift>
Returns 131071.
