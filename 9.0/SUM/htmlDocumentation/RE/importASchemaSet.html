Import a schema set from a zip file.

A schema set can be imported into Toolbox packaging it in a zip file. The structure of the file can be any, with multiple subdirectories and files. The only constraint is that all link to schema of the set shall be placed as relative to the zip root.

So, if schema a.xsd, placed on the root, imports the schema b.xsd that is placed under the directory "gml" the include statement shall contain the following location: "gml/b.xsd".

If this rule is not followed the schema set will not be managed in the right way.
