Sun Java SE Development Kit

The TOOLBOX has been tested** using the following configuration:

  • The Java SE Development Kit (JDK 5.0) + Tomcat 5.5.9

The TOOLBOX has been tested using the package reported above. Anyway more recent versions of the Java 2 SDK can be used, depending on the Tomcat requirements.

(only for Linux) The TOOLBOX has been developed and tested using the Red Hat distribution. Other distributions can have problems during the JDK installation. These problems can be avoided by using more recent versions of the Java 2 SDK.

** Full tests on Linux Red Hat, partial tests on Windows platforms.
