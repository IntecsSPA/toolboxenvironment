Using Jelly

Since TOOLBOX 4.2, any Jelly script can be easily integrated in the TOOLBOX scripts by means of the TOOLBOX XML language tag "jelly".

Jelly scripts can be inserted directly in the TOOLBOX XML script or loaded from files (still from the TOOLBOX XML script). In this regard, please see the tag "jelly" documentation.

The TOOLBOX is released only with the core Jelly tag library, thus only Jelly core tags can be used in the TOOLBOX XML scripts.

To use other Jelly Tag libraries in the TOOLBOX XML scripts perform the following steps:

  1. Download the java archive ".jar" related to the Tag library to use
  2. Copy the file ".jar" in <TOMCAT_HOME>/webapps/TOOLBOX/WEB-INF/lib/
  3. Reload the TOOLBOX pointing the browser to http://<TOMCAT_HOST>[:<TOMCAT_PORT>]/manager/reload?path=/TOOLBOX (filling the possibly prompt window with user name and password chosen when installing Tomcat).

Understanding of the Jelly scripting language is outside of the scope of this tutorial. For further information, please visit the Jelly documentation at "http://jakarta.apache.org/commons/jelly/".
