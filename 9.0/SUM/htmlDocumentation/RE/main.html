Getting started (version 9.0)

In the run-time environment, the TOOLBOX is a configurable application that helps the Service Provider to easily convert its service in SOAP based service without code development.

See the overview page for more details.

New in this release

  • Added support for the deployment of the following OGC catalogues interfaces:
    • EO Products Extension Package for ebRIM (ISO/TS 15000-3) Profile of CSW 2.0 (0.1.9), 06-131r4 and higher
  • Added the support for the integrationof legacy catalogue to be exposed as OGC catalogues interfaces:
    • EO Products Extension Package for ebRIM (ISO/TS 15000-3) Profile of CSW 2.0 (0.1.9), 06-131r4 and higher
    • OGC Cataloguing of ISO Metadata (CIM) using the ebRIM profile of CS-W (0.1.10), 07-038 and higher
  • Added the the possibility to expose an OGC ebRIM CIM catalogue as an INSPIRE compliant catalogue by means of a facade handling the conversion between the two catalogue interfaces.
  • Added the support of the OGC Web Processing Service specification (SOAP and HTTP POST bindings, HTTP GET is under development).
    • Custom processing services integrated via shell scripts
    • GRASS processing engine supported
  • Added the possibility to publish the results of a processing/service on different servers: FTP (built-in), HTTP (built-in), WMS, WFS and WCS (via GeoServer)
  • Added support OGC 07-118r8 "User Management Interfaces for Earth Observation Services" for Toolbox services.
  • Added support OGC 07-118r8 "User Management Interfaces for Earth Observation Services" for external services (PEP).
  • Updated the Toolbox GUI.
  • Added a dock bar for an easier navigation.
  • Updated the Toolbox Gis Client.
