Re-configuring an operation

To re-configure an operation follow these steps.

Display the list of the Service operations available.

Click on the Configure link,the browser should display two different pages depending on the type of operation (synchronous or asynchronous) you want to re-configure.

To re-configure an operation modify the parameters you want to change (for a description of each parameter see the "Adding a Synchronous operation" section for synchronous operations configuration and "Adding an Asynchronous operation" section for asynchronous operations reconfiguration) then click on "Configure".

Moreover the re-configuration pages allow the possibility to display the content of the operation script. To see the content of the operation script click on the "View" button in the row of the script you want to display. The browser will display a text area showing the script. To remove this area, click on the "Close" button.
