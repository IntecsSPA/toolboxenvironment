Removing the Schema

To remove a Schema from the service scope, go to the create/configure page and perform the following steps:

  • Select the "check box" associated to the Schema you want to remove.

  • Click on the "Remove Selected Schema" button. The browser will show a confirmation message.

  • Click on the "Yes" button to remove the selected Schemas from the Service Schema list.

The browser should now display the updated list.

If you select the Service Schema "check box" all the Schema will be removed.

If you select the an SSE Schema "check box" all the SSE Schema will be removed.
