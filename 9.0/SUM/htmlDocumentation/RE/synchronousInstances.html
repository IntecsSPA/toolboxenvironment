Listing synchronous instances

To view the list of the asynchronous instances of the selected service click on the "Display Synchronous instances" section link in the in the "Monitoring center" page. The browser should display a table listing all the service instances

The table displayed includes the following information:

  • TOOLS: this column allows deleting old instances. To delete an old instance click on the "DELETE" link.
  • instance: identifier of the instance. This is a unique key assigned by the TOOLBOX.
  • OrderId: SSE order identifier. It is displayed only if the message is a SSE compliant message.
  • Operation: name of the operation.
  • Date: date in which the request has been received and served by the service.
  • Status: status of the request. Each request can be in one of the following status:
    • executing: the TOOLBOX is executing the First script corresponding to the request.
    • error: the request execution returned an error.
    • completed: this status is reached when the TOOLBOX succeeds in sending back the response
Additionally the View Status page allows sorting the table by clicking on the header's links.
