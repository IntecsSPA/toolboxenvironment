Installing the TOOLBOX

Before installing the TOOLBOX you have to install the following component: JDK and optionally Apache.

The TOOLBOX can be easily installed using a graphical installer. The installer is Java based and can be run on any operating system where a Java virtual machine is available. Depending on the operating system, it can be launched by a double-click or a simple java -jar installer.jar from a shell. The main benefit of the installer is that it provides a clean and unique way of installing the TOOLBOX on operating systems. The installer is reported to run on:

  • Unix-like systems, mostly Linux and FreeBSD variants
  • Windows XP, Vista.

Moreover for each operating system the installer is provided in two versions:

  • Full installer: it includes all the software needed to install the TOOLBOX in the jar file.
  • light installer: it includes only the installer. The resources needed to install the TOOLBOX (TOMCAT and the TOOLBOX jar file) are downloaded at installation time.

In both cases the installer allows:

  • installing the TOOLBOX on a TOMCAT server already installed on the target machine (a check of the TOMCAT version is done at installation time)
  • installing the TOOLBOX on the TOMCAT server included in the installation pack.

Moreover the installer allows upgrading an already installed TOOLBOX to the new version.
