Tutorials

The following tutorials are available:

  • Deploying a Web Processing Service based on GRASS: this tutorial explains how to create a Web Processing Service using GRASS as processing engine.
  • Deploying a Web Processing Service based on Shell script: this video explains how to create a Web Processing Service invoking some processing function via command line.
  • Creating a stand alone EOP catalogue: this video explains how to set up an Earth Observation Catalogue service compliant with the OGC-06-131r6 interface (EOP EbRIM catalogue) .
  • Securing a Service: this video explains how to secure a service deployed on the Toolbox.
  • Creating a security Catalogue Proxy: this tutorial explains how to create a security proxy (Policy enforcement Point) for a catalogue service deployed on a remote server.
