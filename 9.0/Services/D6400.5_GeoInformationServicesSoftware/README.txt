services_TBX
------------
Contains backups of TOOLBOX Web Processing Services
These services call the perl scripts in the services_perl directory
The Software User Manual contains further information on how to import these services into the TOOLBOX 

services_perl
-------------
Contains perl scripts, template and auxiliary files implementing the Web Processing Services
The Software User Manual contains more information on how to install these files


These services were developed in the frame of the EC FP7 GENESIS Project by GIM nv.
