addOrderConfirmation

Tag Name addOrderConfirmation
Attributes
Name Type Use Default Description
mnemonicId xsd:string required   A mnemonic value to be linked to the order ocnfirmation.
Description This tag add an order confirmation entry for the service. The order confirmation id is returned by the tag.Mnemonicid specifies an attribute useful for finding the order confirmation from the web page. It can be used for linking the confirmation to the portal orderId.
Examples 
<addOrderConfirmation mnemonicId="$orderId" />
This tag adds an order confirmation for the service that is executing it. The value of the variable orderId is linked to this confirmation request and displayed to the administrator on the management web page.

 

checkOrderConfirmation

Tag Name checkOrderConfirmation
Attributes
Name Type Use Default Description
id xsd:string required   Identifier of the order confirmation, returned by the addOrderConfirmation tag,
Description This tag returns the status of the order, whose id is specified by the id attribute. If the order has been confirmed by the administrator a true value is returned, false otherwise.
Examples 
<if>
	<oeq>
	    <checkOrderConfirmation id="$orderConfId" />
	    <literal type="boolean" value="true" />
	</oeq>
						
	<setVariable name="returnValue">
		<literal type="boolean" value="true"/>
	</setVariable>
	<setVariable name="returnValue">
		<literal type="boolean" value="false"/>
	</setVariable>
</if>
This tag sets the variable returnValue to true or false depending on the status of the confirmation request whose id is stored by the variable orderConfigId.

getOrderError

Tag Name getOrderError
Attributes
Name Type Use Default Description
id xsd:string required   Id associated to an order confirmation whose error message shall be returned.
Description This tag returns, if any, the error message associated to the confirmation with the provided id. The return value is of type String
Examples 
<getOrderError id="$id"/>
This tag returns, if any, the error message associated to the confirmation with the provided id.
