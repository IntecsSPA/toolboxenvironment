Getting started (version 8.0)

In the run-time environment, the TOOLBOX is a configurable application that helps the Service Provider to easily convert its service in SOAP based service without code development.

Using the TOOLBOX the new Web Services can be integrated within the SSE framework.

See the overview page for more details.

The TOOLBOX allows the possibility of deploying a SOAP interface not compliant with the SSE ICD or the EOLI-XML ICD. Thus it can also be employed outside the context of the SSE framework.

New in this release

  • Added support for OGC catalogues interfaces:

    • OGC-06-131r OGC Catalogue Services Specification 2.0 Extension Package for ebRIM (ISO/TS 15000-3) Application Profile
    • OGC-07-038 OGC Cataloguing of ISO Metadata (CIM) Using the ebRIM profile of CS-W
    • OGC-07-045
  • Integrated ebRR ebRIM catalogue as internal instance.
  • Added administration pages for editing GetCapabilities, harvesting data and creating catalogue database.
  • Added support to security enforcement for Toolbox services.
