Strings

Tag Name atoi
Description Returns an integer obtaining converting the String returned by the enclosed element.
Examples 
<atoi>
	<string>33</string>
</atoi>
Returns an integer valuing 33. 
<atoi>
	<stringCat>
		<string>3</string>
		<string>4</string>
	</stringCat>
</atoi>
Returns an integer valuing 34.

Tag Name atof
Description Returns a float obtained converting the String returned by the enclosed element.
Examples 
<atof>
	<string>33.6</string>
</atof>
Returns a float with value 33,6.

Tag Name itoa
Description Returns a string obtaining converting the integer returned by the enclosed element.
Examples 
<itoa>
	<literal value="33"/>
</itoa>
Returns the string "33". 
<itoa>
	<divide>
		<literal value="40"/>
		<literal value="10"/>
	</divide>
</itoa>
Returns the string "4".

Tag Name print
Attributes
Name Type Use Default Description
newLine xsd:boolean true
Flag indicating whether a newLine must be placed at the end of the string to print (default) or not ("newLine" set to "false")
Description Prints a string on video, with an optional newLine (newLine attribute).
Examples 
<sequence>
	<print newLine="false">
		<string>Hello </string>
	</print>
	<print>
		<string>World!</string>
	</print>
</sequence>
Prints on the standard output (Tomcat log file) the string "Hello World!" 
<sequence>
	<print>
		<string>Line1</string>
	</print>
	<print>
		<string>Line2</string>
	</print>
</sequence>
Prints on the standard output (Tomcat log file) the lines:
Line1
Line2

Tag Name randomString
Attributes
Name Type Use Default Description
length xsd:positiveInteger required
The length of the string to be produced
Description Produces a random string using lower and upper case letters, and digits. The length of the string is given by the length attribute.
Examples 
<randomString/>
Produces a random string 
<randomString length="8" />
Produces a random string of length 8.

Tag Name string
Description Returns the string given as text inside the tag.
Examples 
<string>This is a string</string>
Produces the string "This is a string".

Tag Name stringCat
Description Returns a string obtained concatenating the strings returned by each of the enclosed elements. The enclosed elements can return any object, as they are converted to a string using the String.valueOf method.
Examples 
<stringCat>
	<string>My age is </string>
	<literal value="33"/>
	<string>.</string>
</stringCat>
Returns the string "My age is 33."

Tag Name stringLength
Description Returns the length of the string returned by the element enclosed in this element.
Examples 
<stringLength>
	<string>This is a string</string>
</stringLength>
Returns the integer 16.
