Test center

The Test center page provides four different test tools:
  • Send a SOAP message : This page allows testing a deployed service directly from the TOOLBOX Web Application. The service can be deployed locally or in a remote TOOLBOX installation.
  • Test a Catalogue : This page allows testing a catalogue exposing a SOAP interface compliant with the SSE or EOLI ICD.
  • Validate an XML script : This page allows validating an XML script describing the service behavior. The script is validated against the XML scripting Schema and the result of the validation is reported to the user.
  • Push server : The TOOLBOX provides a SOAP end point (i.e. Push Server) which can be used to test asynchronous operation. This section allows listing and managing the messages pushed back to the "Push Server".
