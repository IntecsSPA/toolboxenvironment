TOOLBOX Services access via the SSE Portal

The AOIFetcher.jar contained in the present directory must be installed on the
TOOLBOX, i.e. it must be placed at <TOMCAT_ROOT>/webapps/TOOLBOX/WEB-INF/lib
and the TOOLBOX must be reloaded pointing the browser to:

http://<TOMCAT_HOST>:<TOMCAT_PORT>/manager/reload?path=/TOOLBOX (if promped
provide the administrator user name and password chosen during the Tomcat installation

NOTE: Access the SSE Portal and perform an Order using the parameters contained in the TS_21_01 section
