Hardware requirements

The ARMS does not require any special hardware to run, except for those requirements dictated by the software used for running it (Java, Tomcat,Toolbox). See software user manuals for their hardware requirements. The host configuration will vary depending on required storage area, responsiveness and deployment configuration. A typical configuration is the following:
  • I586 Intel processor
  • 2 GB of RAM
  • 250 GB of free disk space
