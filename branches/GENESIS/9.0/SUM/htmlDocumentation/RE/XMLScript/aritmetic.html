Arithmetic Operator

Tag Name dec
Description Decrements by one the enclosed LValue. LValue can be one of variable, field and arrayLocation tags. The returned value is the value after decrement. (Pre-decrement)
Examples 
<dec>
	<variable name="aVariable"/>
</dec>

The variable named "aVariable" holds an integer: its value is decremented and returned. 

<dec>
	<arrayLocation>
		<variable name="arrayName"/>
		<variable name="arrayIndex"/>
	</arrayLocation>
</dec>

The value contained in the array named "arrayName" at the index "arrayIndex" is decremented by one and returned 

<dec>
	<field fieldName="anObjectField">
		<object>
			<variable name="anObject"/>
		</object>
	</field>
</dec>

The value contained in the field named "anObjectField" of the object named "anObject" is decremented by one and returned.

<dec>
	<field fieldName="obiectField">
		<class>
			<classLiteral class="package.name.ClassName"/>
		</class>
	</field>
</dec>
The value contained in the static field named "aField" of the class named "ClassName" of the package "package.name" is decremented by one and returned.

Tag Name divide
Description Returns the integer division of the integer expressions returned by the enclosed elements. If there is no enclosed tag it returns 1. If there is one element it is returned as is. If there are at least two elements, all the expressions returned by the second and subsequent elements are used as divisors.
Examples 
<divide/>
Returns 1. 
<divide>
	<literal value="1024"/>
</divide>
Returns 1024. 

<divide>
	<literal value="1024"/>
	<atoi>

        <string>8</string>

    </atoi>
</divide>

Returns 128.


<divide>
	<literal value="1024"/>
	<atoi>

        <string>8</string>

    </atoi>

    <variable name="variableName"/>
</divide>
1024 is divided by 8, the result (128) is divided by the value stored in the variable named "variableName".

Tag Name minus
Description Returns the difference of the integer expressions returned by the enclosed elements. If there is no enclosed tag it returns 0. If there is one element it is returned as is. If there are at least two elements, all the expressions returned by the second and subsequent elements are subtracted from the first.
Examples 
<minus/>
Returns 0. 
<minus>
	<literal value="1024"/>
</minus>
Returns 1024. 

<minus>
	<literal value="1024"/>
	<atoi>

        <string>8</string>

    </atoi>
</minus>

Returns 1016.


<minus>
	<literal value="1024"/>
	<atoi>

        <string>8</string>

    </atoi>

    <variable name="variableName"/>
</minus>
1024 is decremented by 8, the result (1016) is decremented by the value stored in the variable named "variableName".

Tag Name module
Description Returns the module of the two integer expressions returned by the enclosed elements.
Examples 
<module>
	<literal value="23"/>
	<atoi>

        <string>8</string>

    </atoi>
</module>
Returns 7.

Tag Name multiply
Description Returns the product of the integer expressions returned by each of the enclosed elements. If there is no enclosed element, it returns 1.
Examples 
<multiply/>
Returns 1. 
<multiply>
	<literal value="1024"/>
</multiply>
Returns 1024. 

<multiply>
	<literal value="1024"/>
	<atoi>

        <string>8</string>

    </atoi>
</multiply>

Returns 8192.


<multiply>
	<literal value="1024"/>
	<atoi>

        <string>8</string>

    </atoi>

    <variable name="variableName"/>
</multiply>
1024 is multiplied by 8, the result (8192) is multiplied by the value stored in the variable named "variableName".

Tag Name negate
Description Returns the integer returned by the enclosed element, changing sign.
Examples 
<negate>
	<literal value="1024"/>
</negate>

Returns -1024.

<negate>
	<literal value="-1024"/>
</negate>
Returns 1024.

 

Tag Name plus
Description Returns the sum of the integer expressions returned by each of the enclosed elements. If there is no enclosed tag it returns 0.
Examples 
<plus/>
Returns 0. 
<plus>
	<literal value="1024"/>
	<atoi>

        <string>8</string>

    </atoi>
</plus>

Returns 1032.


<plus>
	<literal value="1024"/>
	<atoi>

        <string>8</string>

    </atoi>

    <variable name="variableName"/>
</plus>
1024 is incremented by 8, the result (1032) is incremented by the value stored in the variable named "variableName".
