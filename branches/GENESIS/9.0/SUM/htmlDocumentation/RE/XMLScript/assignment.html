Assignment Operators

Tag Name set
Description Sets the LValue given by the first enclosed element, with the value given by the second. LValue can be one of variable, field, arrayLocation
Examples 
<set>
	<variable name="aVariable"/>
	<literal value="0"/>
</set>
Sets to 0 the value stored in the variable "aVariable". 
<set>
	<arrayLocation>
		<variable name="arrayName"/>
		<variable name="arrayIndex"/>
	</arrayLocation>
	<multiply>
		<variable name="aVariable"/>
		<literal value="2"/>
	</multiply>
</set>
The location a index "arrayIndex" of the array "arrayName" is set to the value of the variable "aVariable" multiplied by 2. 
<set>
	<field fieldName="anObjectField">
		<object>
			<variable name="anObject"/>
		</object>
	</field>
	<object>
		<variable name="anOtherObject"/>
	</object>
</set>
The field "anObjectField" of the object "anObject" is set with the object "anOtherObject".

Tag Name setVariable
Attributes
Name Type Use Default Description
name xsd:string The name of the variable to be set.
Description Sets the variable called as the name attribute with the value given by the enclosed element
Examples 
<setVariable name="aFile">
	<newObject>
		<classLiteral class="java.io.File"/>
		<parameter>
			<classLiteral class="java.lang.String"/>
			<variable name="filePath"/>
		</parameter>
	</newObject>
</setVariable>
The variable named "aFile" is set to the reference to the File whose path is stored in the variable "filePath". 
<setVariable name="anXMLDocument">
	<loadXML>
		<string>/home/toolbox/file.xml</string>
	</loadXML>
</setVariable>

The variable "anXMLDocument" is set to the XML Document contained in the file "file.xml".


NOTE: all variables stored with <setVariable name="VariableName"> can be later retrieved with <variable name="VariableName"/>

WARNING: the following case-sensitive string are XML TOOLBOX script language reserved words and cannot be used as variable names:

  • orderId
  • massHost
  • operation
  • operationMode
  • serviceMode
  • xmlRequest
