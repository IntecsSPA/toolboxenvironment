Date and Time

Tag Name now
Description This tag returns the current time as a date Object.
Examples 
<now/>

The current time is returned.

Tag Name isBefore
Description This tag is used to check if one date (first child item) comes temporarily before another (passed as second child item). The returned value is true if the first date comes before the second one, false otherwise.
Examples 
<isBefore>
    <now/>
    <variable name="anotherDate"/>
</isBefore>


The tag returns true if anotherDate is a future date.
   

Tag Name dateStringFormatter
Attributes
Name Type Use Default Description
format xsd:string required   The format of the date. See here for formatting a date.
Description This tag is used to return a date type as a string.
Examples 
<dateStringFormatter format="ddMMyyyyHHmmssSSS">
   <now />
</dateStringFormatter>

The current time is returned as string, formatted as the following: 29062007134507259

Tag Name date
Attributes
Name Type Use Default Description
format xsd:string required   The format of the date. See here for formatting a date.
Description This tag is used to return a string as a date.
Examples 
<date format="ddMMyyyyHHmmssSSS">
   <string>11052007120000000</string>
</date>


This tag returns a date object containing the date 11 May 2007 12:00:00

 

Tag Name timeDifference
Description This tag is used to calculate the difference between two dates. The difference is returned in milliseconds.
Examples 
<timeDifference>
   <date format="ddMMyyyyHHmmssSSS">
       <string>11052007120000000</string>
   </date>
   <now/>
</timeDifference>
