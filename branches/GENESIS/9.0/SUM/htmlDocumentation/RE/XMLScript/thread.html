Threading

Tag Name fork
Description This tag creates a new thread for executing Toolbox scripts. The script to be executed can be provided through the file tag, giving its full path, or simply putting a script snippet as a child of the fork tag.
Examples 
<fork>
		<sequence>
	    	<ftpGet transfer="binary">
	        	<host>
		        	<string>pisa007.pisa.intecs.it</string>
		        </host>
	        	<port>
		        	<literal value="21" />
	        	</port>
	        	<user>
		        	<string>dvd</string>
	        	</user>
	        	<password>
		        	<string>intecs</string>
	        	</password>
	        	<remotePath>
		        	<string>/home/ftpaccount/wtp-all-in-one-sdk-R-1.5.1-200609230508-win32.zip</string>
		        </remotePath>
	         	<localPath>
		         	<stringCat>
		              	<string>/tmp/downloadedFile.zip</string>
			        </stringCat>
	         	</localPath>
        	</ftpGet>
   	</sequence>
</fork>

This piece of code executes asynchronously an FTP file transfer.

<fork>
     <file>/home/toolbox/myAsynchScript.tscript</file>
</fork>

This piece of code executes asynchronously the script stored in the file myAsynchScript.tscript.
