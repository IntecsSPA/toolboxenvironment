Importing a service

To import a service follows these steps.

Select the “Service Management” link on the top bar menu then select the “Service Manager” section

Select the Import Services tab

Select a Service Group zip file previously exported and click on Import Services. The list of the services included in the zip file is displayed

Select the services you want import. If the service is already present in the system the interface will notify this with a red sign.

Select a name that is not present in the system then click on Import Services.

A pop up window will report the result of the operation.
