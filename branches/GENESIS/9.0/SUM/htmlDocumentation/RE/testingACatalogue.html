Testing a Catalogue

Click on the "Test center" link on the toolbar. The browser should display

 

Click on the "Test a catalogue" section link.

This page allows testing a catalogue exposing a SOAP interface compliant with the SSE ICD and with the EOLI ICD

 

Select the operation you want to test from the list and click on the "Next>>" button.

If a Search operation has been selected the following page should be displayed

 

Fill the "Query parameters" and the "SOAP call parameters".

Push the "send" button. The browser should display the response page.

If a Present operation has been selected the following page should be displayed

 

Fill the "Query parameters" and the "SOAP call parameters".

Push the "send" button. The browser should display the response page.
