Update a WPS operation

To update a WPS operation follow these steps

Select a WPS service and click on the Operations management menu link. A page reporting the operations available for the selected service should be displayed.

Click on the Resource Manager button. A pop up windowshould appear

To update the Describe Process document select the Edit icon in the Describe Process row. The current Describe Document is loaded in the edit area

Update the document and click on the Update WPS Processing Describe Information button. If the document is correct the describe information is updated and the framework recreate all the resources needed for the service.

: if you update the describe some variable taht you are using in the processing script may be no more available. Thus make sure to maintain the describe and the processing script aligned.

To update the processing script select the Edit icon in the script row (it could be one of the available processing engine script e.g shell script, GRASS script or Toolbox script).

The current processing script is loaded in the edit area

Update the document and click on the Change WPS Processing Engine Scriptbutton.
