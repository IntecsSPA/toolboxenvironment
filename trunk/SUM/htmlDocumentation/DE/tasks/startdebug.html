Start debug of a script

The debug session can be started, selecting one of the available launch configurations (click here to know how to create one) on the menu that appears clicking on the arrow on the right of the debug icon (represented on the IDE as a green bug).

Clicking on the launch configuration, the debug session is immediately started. If the current perspective is not the debug one, it is possible to move to it, clicking on the perspective icon on the right of the IDE.
