e-mail

Tag Name sendMail
Attributes
Name Type Use Default Description
smtpServer xsd:string required The smpt server address to be used
Description Sends an e-mail. The smpt-server attribute gives the smpt server address to be used. The from child contains a child that must return the mail address to be set as sender of the e-mail. The to child contains a child that must return the mail address of the recipient of the e-mail. The otherRecipients element contains zero or more to, cc or bcc children each containing a child that must give additional recipient e-mail addresses. The subject child contains a child that must return the subject of the e-mail. The text child contains a child that must return the text of the e-mail. Any subsequent attachment element cause an attachment to be added to the e-mail. The name attribute (optional) of such elements gives the name to be given to the attchment in the e-mail. The element enclosed in the attachment element gives the path of the file to be attached.
Examples 
<sendMail smtpServer="smtpServerAddress">
	<from><string>bb@pisa.intecs.it</string></from>
	<to><string>pn@pisa.intecs.it</string></to>
	<otherRecipients>
		<cc><string>sg@pisa.intecs.it</string></cc>
	</otherRecipients>
	<subject><string>Required file</string></subject>
	<text><string>Dear Patrizia,.....</string></text>
	<attachment>
		<string>C:\fileToBeAttached.xml</string>
	</attachment>
</sendMail>
Sends an e-mail from "bb@pisa.intecs.it" to "pn@pisa.intecs.it" (putting in cc sg@pisa.intecs.it) using the smtp server "smtpServerAddress". The e-mail has attached the file "C:\fileToBeAttached.xml" and its content is: "Dear Patrizia,...".
