Tutorial

This tutorial is aimed at helping service providers to integrate their services into SSE using the TOOLBOX. Understanding of the tutorial requires at least basic knowledge of:

  • XML language
  • XML schemas
  • XSL Transformations

This tutorial does NOT cover the whole process required to integrate a service within SSE: on this subject, please see the SSE ICD.

Moreover it provides the description of some script used to integrate different services in the SSE framework (the VITO Services - a file exchange based service, the GIM ICE Services - a HTTP based service and the ACS Catalogue - a JDBC based catalogue)
